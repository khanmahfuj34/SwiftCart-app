User Action → State Update → Animation → Navigation → Auto-Close

[Menu Tap]
    │
    ▼
setDrawerVisible(true)
    │
    ▼
Animated.timing(
  slideAnim: -300 → 0,
  duration: 300ms
)
    │
    ▼
Drawer visible with overlay
    │
    ▼
[Item Click] → router.push(route)
    │
    ▼
Navigation completes
    │
    ▼
onClose() → setDrawerVisible(false)
    │
    ▼
Animated.timing(
  slideAnim: 0 → -300,
  duration: 300ms
)
    │
    ▼
Drawer hidden

## Color Scheme

```
LIGHT MODE:
├─ Background: #FFFFFF (White)
├─ Text Primary: #0F172A (Navy)
├─ Text Secondary: #6B7280 (Gray)
├─ Active Background: #F3F4F6 (Light Gray)
├─ Badge: #EF4444 (Red)
├─ Icons: #6B7280 (Gray) → Active: #0F172A
└─ Overlay: rgba(0, 0, 0, 0.5)

DARK MODE:
├─ Background: #1A1A1A (Dark Gray)
├─ Text Primary: #FFFFFF (White)
├─ Text Secondary: #AAAAAA (Medium Gray)
├─ Active Background: #2A2A2A (Lighter Gray)
├─ Badge: #EF4444 (Red) - Same
├─ Icons: #888888 → Active: #FFFFFF
└─ Overlay: rgba(0, 0, 0, 0.7)
```

## Animation Timings

```
Drawer Open:
├─ Duration: 300ms
├─ Easing: Linear
├─ Transform: translateX(-300 → 0)
└─ useNativeDriver: true

Drawer Close:
├─ Duration: 300ms
├─ Easing: Linear
├─ Transform: translateX(0 → -300)
└─ useNativeDriver: true

Menu Item Highlight:
├─ Duration: Instant
├─ Background change
└─ Active state applied

Navigation:
├─ Duration: 100-200ms
├─ Depends on screen load
└─ Auto-close on complete
```
